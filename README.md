# Rock-Paper-Scissor
RPS using JavaScript

RPS in numbers to easily apply script
```
var you = 0
var enemy = rnd_number

if (you - enemy == 1 || you - enemy == -2) // Win
else if (you == enemy) // Draw
else // Lose
```
0 - 0 = 0   // Draw
0 - 1 = -1  // Lose
0 - 2 = -2  // Win
1 - 0 = 1   // Win
1 - 1 = 0   // Draw
1 - 2 = -1  // Lose
2 - 0 = 2   // Lose
2 - 1 = 1   // Win
2 - 2 = 0   // Draw
